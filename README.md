# Hospital Admissions Dashboard

This repository contains an interactive dashboard and supporting outputs for visual analysis of hospital admissions data.

## Main file

- `index.html`: self-contained interactive dashboard for GitHub Pages

## Supporting files

- `dashboard.html`: same dashboard file as a named copy
- `code.py`: data processing and dashboard generation script
- `cleaned_summary_categories.csv`: cleaned category-level summary table
- `chapter_timeseries.csv`: chapter-level time series table

## Notes

- The dashboard focuses on age-group change over time using small-multiple heatmaps.
- The published dashboard is self-contained and can be viewed directly in a browser.
